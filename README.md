# Симулятор мобильного робота с лидаром

Настольное приложение на WPF. Робот движется по карте с препятствиями. Лидар сканирует пространство лучами и строит карту на основе полученных точек. Интерфейс отображает траекторию робота, облако точек от лидара, скорость и текущий угол поворота.

## Основные возможности

- Карта с препятствиями, заданная разработчиком или загружаемая из файла.
- Модель движения робота с управлением скоростью и углом поворота.
- Модель лидара: набор лучей, расчет пересечений со стенами, определение расстояний.
- Построение карты по данным лидара в реальном времени.
- Отображение траектории движения робота.
- Управление роботом с клавиатуры или через панель кнопок.
- Возможность паузы, сброса карты, изменения скорости симуляции.

## Технологии

- WPF, C#, .NET 8.
- MVVM для разделения логики и интерфейса.
- Отрисовка сцены через Canvas или WriteableBitmap.
- Матрицы поворота и координатные преобразования для модели робота и лучей лидара.
